Made 100% in Processing 3.4 (Java)

This HTML viewer is pretty decent
It can tell you how many characters are in a line
and shows line numbers
best part, you can just search a webpage! You don't have to download
the html for a page to see it's raw HTML anymore!
You might even find stuff
you didn't expect to see!
Like comments in HTML explaining what something does
or even things that were just commented out such as
scrapped ideas or unfinished features.
This is not really a multi-tool,
it's like a browser but it can display
the HTML only, it changes title based off
the title tag... if it isn't multi-lined or embedded in some other tag.
*Not really, it's just kind of an option
